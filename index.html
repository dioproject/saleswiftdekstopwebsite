Point of Sale & Pembayaran

Kasir cepat dengan berbagai metode pembayaran untuk semua jenis transaksi.

🛒

Keranjang Cepat

Klik produk langsung masuk keranjang. Atur jumlah dengan tombol plus/minus. Diskon per item dan diskon transaksi.

🔍

Pencarian & Barcode

Cari produk berdasarkan nama, SKU, atau scan barcode. Mendukung barcode primer dan sekunder per produk.

💵

Pembayaran Tunai

Masukkan jumlah uang diterima, sistem otomatis hitung kembalian. Cepat dan akurat.

📱

QRIS Statis

Tampilkan QR code merchant. Kasir konfirmasi setelah notifikasi pembayaran diterima.

💳

E-Wallet Multi-Akun

GoPay, OVO, DANA, ShopeePay, LinkAja, dan lainnya. Admin daftarkan banyak akun, kasir pilih dari dropdown.

🏦

Transfer Bank Multi-Rekening

BCA, Mandiri, BRI, BNI, dan lainnya. Admin daftarkan banyak rekening, kasir pilih tujuan transfer.

💰

Diskon & Pajak

Diskon per item (pre-tax) dan diskon transaksi. Pajak per produk dalam basis poin (misal 1100 = 11% PPN).

🍽️

Konteks Pesanan

Untuk cafe/restoran: tipe pesanan (dine-in/takeaway), nomor meja, nama pelanggan. Cetak struk dan tiket dapur.

Produk, Kategori & Stok

Kelola katalog produk, kategori, gambar, barcode, dan stok secara lengkap.

📦

CRUD Produk

Buat, edit, hapus (soft delete) produk. Field lengkap: SKU, nama, deskripsi, harga jual/beli, pajak, satuan, gambar.

🏷️

Manajemen Barcode

Banyak barcode per produk dengan flag primer. Scanner langsung temukan produk yang tepat.

📸

Gambar Produk

Upload gambar produk (maks 2 MB). Validasi MIME type. Tampilkan di POS untuk visual cepat.

📂

Kategori Hierarki

Kategori induk-anak dengan urutan sortir kustom. Filter produk berdasarkan kategori di POS.

📊

Stok & Pergerakan

Tracking stok per produk. Setiap perubahan dicatat: penjualan, refund, pembelian, penyesuaian, stok opname, kerusakan.

⚠️

Alert Stok Rendah

Sistem peringatkan saat stok <= level reorder. Lihat daftar produk yang perlu restok.

📋

Stok Opname

Cocokkan stok sistem dengan stok fisik. Catat sesi opname, hitung selisih, dan riwayat lengkap.

🔒

Blokir Stok Habis

POS otomatis blokir transaksi yang akan menyebabkan stok negatif. Data konsisten selalu.

Pencarian Pelanggan

Poin Loyalitas

Riwayat Transaksi

Keamanan & Lisensi

Sistem keamanan berlapis dan lisensi offline untuk perlindungan aplikasi.

🔐

Argon2id

Password hashing dengan Argon2id — standar keamanan terkini. Tidak ada password plaintext.

🔑

OS Keyring

Lisensi disimpan di OS keyring (bukan plaintext di disk). Keamanan tingkat sistem operasi.

🔒

AES-GCM Encryption

Pengaturan rahasia (API key, token) dienkripsi dengan AES-GCM. Aman di database lokal.

📄

Lisensi Offline

Tidak perlu internet. Lisensi terikat fingerprint perangkat. File .lic dari vendor.

⏰

Grace Period

Setelah expired, aplikasi tetap bisa diakses selama grace period yang dikonfigurasi.

🛡️

Read-Only Mode

Lisensi expired blokir transaksi baru tapi tetap bisa baca riwayat. Data tidak hilang.

🔍

Clock Tamper Detection

Jika jam sistem dimundurkan, status lisensi jadi "tampered". Cegah manipulasi waktu.

💾

Backup & Restore

Backup database SQLite ke file. Restore dari backup. List semua backup dengan ukuran dan waktu.

Cara install

Instalasi sampai aplikasi aktif

Alur dibuat sederhana: download installer, import lisensi, lalu langsung pakai.

1. Download & Install 2. Import .lic 3. Setup Awal

1. Download & Install Aplikasi

Hubungi vendor untuk mendapatkan file installer .msi SaleSwiftDesk.

  1. Download file installer .msi dari vendor.
  2. Klik dua kali file installer untuk memulai instalasi.
  3. Ikuti instruksi sampai selesai.
  4. Buka aplikasi dari Desktop atau Start Menu.
  5. Jika Windows menampilkan konfirmasi keamanan, pilih untuk melanjutkan.

2. Import File Lisensi .lic

Saat pertama kali dibuka, SaleSwiftDesk akan meminta aktivasi. Pilih file lisensi .lic yang diberikan vendor, lalu klik aktivasi.

Catatan: file .lic berbeda untuk setiap klien/perangkat. Simpan file ini baik-baik untuk kebutuhan install ulang.

3. Setup Awal Setelah Aktif

  • Buat akun admin pertama.
  • Isi profil toko, alamat, jenis usaha, dan logo.
  • Aktifkan metode pembayaran yang dipakai toko.
  • Tambah kategori, produk, foto produk, dan stok awal.
  • Atur printer thermal jika ingin cetak struk otomatis.
  • Buat transaksi percobaan untuk memastikan kasir siap digunakan.
